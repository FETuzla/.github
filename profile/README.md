# FETuzla
[https://fet.ba/](https://fet.ba/)

FETuzla is a student led GitHub organization for students of our university to collaborate on open-source projects, learn development skills, and contribute to meaningful software.

---

## Repositories

Each repository is a student-maintained project. Examples:

- **information** – metadata about webpages and contributors.
- **ScheduleGenerator** – generate a custom schedule and import it to your calendar.
- **Exam-calendar-generator** – import exam dates to your calendar.

---

## How to Contribute

1. Fork the repo.
2. Create a branch.
3. Make changes.
4. Open a Pull Request with a clear description.

Follow any repository-specific rules.

---

## Community

- Ask questions in Issues.
- Respect others and follow a professional code of conduct.
- Learn, collaborate, and build a portfolio of work.
